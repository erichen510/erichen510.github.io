基本数据结构

堆和栈的内存区别

堆：公共内存空间，谁分配谁回收。堆上分配的东西（对象、数据），生命周期是全局的。
栈：函数结束，栈就会消失。用于保存线程执行的动作以及数据引用，java中，有一个线程便有一个线程栈与之对应。
常量池： 堆中分配的一块内存，用来存储显示声明的常量。

生成器 迭代器 与 可迭代

生成器与迭代器的区别
可迭代（iterable)：可以直接作用与for循环的对象。
生成器（generator）：可以被next()函数调用，并不断返回
迭代器：迭代器可以支持惰性计算

list,dict,string 都是可迭代，但不是迭代器。
